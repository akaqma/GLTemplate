GLTemplate
============

概要
------
OpenGLの共通する基本部分をまとめておく

環境
-----
- Mac OSX (Lion)
- Xcode 4.5
- OpenCV 2.4.2


内容
-----

- [GLUT] GLUTのみの基本的な記述
    - ![画像1](https://raw.github.com/akaqma/GLTemplate/master/_images/GLUT_1.png "画像のタイトル")
- [GL2CV] OpenGLの描画データをOpenCVのデータとして渡す
- [CV2GL] OpenCVで読み込んだデータをGLでテクスチャとして貼り付ける
